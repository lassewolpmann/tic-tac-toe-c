# Simple tic-tac-toe game written in C
## Build instructions
```shell
cmake --build build/
```

## Run instructions
```shell
build/tic_tac_toe
```
The command expects one argument for the player symbol. Valid input is either x or o.

Play as x:
```shell
build/tic_tac_toe x
```
Play as o:
```shell
build/tic_tac_toe o
```
