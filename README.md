# Simple tic-tac-toe game written in C
## Setup
```shell
mkdir build
cd build/
cmake ../
```
## Build
```shell
cmake --build build/
```
## Run
```shell
build/tic_tac_toe
```
The command expects one argument for the player symbol. Valid input is either x or o.

Play as x:
```shell
build/tic_tac_toe x
```
Play as o:
```shell
build/tic_tac_toe o
```
